Claro, vou mostrar um exemplo básico de como criar um componente React Paralax de estrelas no céu, baseado no projeto do Codepen que você compartilhou.

Antes de começar, você precisa ter o React instalado em seu projeto. Você também precisará de um arquivo CSS que contém o estilo para o efeito de paralaxe.

Aqui está o código do componente React:

javascript
import React, { useState, useEffect } from 'react';
import './estilo.css';

const Estrelas = () => {
  const [scrollTop, setScrollTop] = useState(0);

  useEffect(() => {
    const onScroll = (e) => {
      setScrollTop(e.target.documentElement.scrollTop);
    };
    window.addEventListener('scroll', onScroll);
    return () => window.removeEventListener('scroll', onScroll);
  }, []);

  return (
    <div className="estrelas" style={{ transform: `translateY(${scrollTop}px)` }}>
      {[...Array(150)].map((_, i) => (
        <div className={`estrela estrela-${i}`} key={i} />
      ))}
    </div>
  );
};

export default Estrelas;

Este componente usa o estado do React para armazenar a posição atual do scroll, que é atualizada sempre que o usuário rola a página. Isso é feito usando o hook useState.

O hook useEffect é usado para adicionar e remover o evento de scroll quando o componente é montado e desmontado, respectivamente.

A classe estrelas é aplicada ao contêiner pai das estrelas, e a propriedade transform é usada para aplicar o efeito de paralaxe. A classe estrela é aplicada a cada estrela individual, e sua posição é definida no arquivo CSS.

import React, { useRef, useState, useEffect } from "react";
import Flickity from "flickity";

import "flickity/css/flickity.css";

const Carousel = () => {
  const [flickity, setFlickity] = useState(null);
  const flickityRef = useRef(null);

  useEffect(() => {
    if (!flickity) {
      setFlickity(
        new Flickity(flickityRef.current, {
          // Opções do Flickity
          // selectedAttraction: 0.3,
          // friction: 0.6
        })
      );
    }

    return () => flickity.destroy();
  }, [flickity]);

import { useRef } from "react";
import "tailwindcss/tailwind.css";
import "flickity/css/flickity.css";
import Flickity from "flickity";

const Carousel = ({ children }) => {
  const flickityRef = useRef(null);

  const flickityOptions = {
    prevNextButtons: false,
    pageDots: true,
    wrapAround: true,
    resize: true,
    initialIndex: 0,
  };

  const flickityPrev = () => {
    flickityRef.current.previous();
  };

  const flickityNext = () => {
    flickityRef.current.next();
  };

  const flickityRestart = () => {
    flickityRef.current.select(0);
  };

  return (
    <div className="screen bg-blue-600 flex flex-col justify-center items-center border-2 border-gray-800 max-w-3xl mx-auto rounded-lg shadow-lg">
      <div
        className="carousel"
        ref={flickityRef}
        options={flickityOptions}
      >
        {children}
      </div>
      <div className="flex justify-center items-center mt-4">
        <button
          className="button mr-4"
          onClick={flickityPrev}
        >
          Anterior
        </button>
        <button
          className="button mr-4"
          onClick={flickityNext}
        >
          Próximo
        </button>
        <button
          className="button"
          onClick={flickityRestart}
        >
          Reiniciar
        </button>
      </div>
    </div>
  );
};

export default Carousel;

Para transformar a galeria de fotos em um modo carrossel, podemos utilizar a biblioteca react-slick que é especialmente desenvolvida para esse propósito. Aqui está um exemplo de como fazer:

javascript
import React from 'react';
import Slider from 'react-slick';
import 'slick-carousel/slick/slick.css';
import 'slick-carousel/slick/slick-theme.css';
import img1 from '../../assets/images/img1.jpg';
import img2 from '../../assets/images/img2.jpg';
import img3 from '../../assets/images/img3.jpg';
import img4 from '../../assets/images/img4.jpg';
import img5 from '../../assets/images/img5.jpg';
import img6 from '../../assets/images/img6.jpg';

export default function Gallery() {
  const settings = {
    dots: true,
    infinite: true,
    speed: 500,
    slidesToShow: 3,
    slidesToScroll: 1
  };

  return (
    <Slider {...settings}>
      <div>
        <img src={img1} alt="img1" />
      </div>
      <div>
        <img src={img2} alt="img2" />
      </div>
      <div>
        <img src={img3} alt="img3" />
      </div>
      <div>
        <img src={img4} alt="img4" />

      </div>
      <div>
        <img src={img6} alt="img6" />
      </div>
    </Slider>
  );
}

Nesse exemplo, estamos importando as imagens e utilizando o componente Slider do react-slick para criar o carrossel. Configuramos algumas propriedades como dots (para indicar as imagens), infinite (para que o carrossel possa continuar passando as imagens infinitamente), speed (para definir a velocidade da transição entre as imagens), slidesToShow (para definir quantas imagens devem aparecer ao mesmo tempo) e slidesToScroll (para definir quantas imagens devem ser passadas ao clicar nas setas de navegação).
